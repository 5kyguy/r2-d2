# R2-D2

R2-D2 is my own config and setup on top of **Arch Linux** — a modern, opinionated desktop and tooling layer. It installs via a single curl command (no ISO). Use it for fresh installs, repairs, or a full reset:

```bash
curl -fsSL https://raw.githubusercontent.com/5kyguy/r2-d2/refs/heads/master/boot.sh | bash
```

R2-D2 was originally forked from [Omarchy](https://github.com/basecamp/omarchy) and has since diverged into its own distribution.
