# R2-D2

R2-D2 is my own config and setup on top of **Arch Linux** — a modern, opinionated desktop and tooling layer. It was forked from [Omarchy](https://github.com/basecamp/omarchy) (at commit [8e88b78](https://github.com/basecamp/omarchy/commit/8e88b780d3add21e94ca2f16c9eef5e3b8b35285)). Omarchy ships both a curl-based install and an ISO; this project uses **only the curl method** (no ISO). Install, update, and repair with:

```bash
curl -fsSL https://raw.githubusercontent.com/5kyguy/artoo-d2/refs/heads/dev/boot.sh | bash
```
